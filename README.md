### Web

Step 0: Install Node.js

Step 1: Install PNPM by NPM

```shell
npm i pnpm -g
```

Step 2: Install dependencies

```shell
cd web
pnpm i 
```

Step 3: Run the project
```shell
pnpm run dev
```
